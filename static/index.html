OPENi

Open-Source, Web-Based, Framework for Integrating Applications with Cloud-based Services and Personal Cloudlets.

What is OPENi?

The OPENi core platform maximises user control by providing an OAuth compliant secure fine-grained access control mechanism, through which, users can control access to their personal data within their Personal Cloudlet and other cloud-based services. Additionally, OPENi aims to increase the end-user's awareness of how their data is accessed through its innovative user-dashboards. OPENi also simplifies integration with 3rd party cloud based services through its API aggregator, its client side library, and service enablers; all of which allow for rapid application development.

Personal Data

OPENi stores user data in Personal Cloudlets. The Cloudlet is built up from a user interataction with OPENi enabled apps. It is an easy to use free storage facility that affords them full control over the storage and sharing of their personal data.

Graph API's

APIs are what connect us, but they can be difficult to manage. In OPENi we abstract this inherent volitility in dealing with cloud based services by providing our own simple community managed Graph API, greatly increasing the power and reach of OPENi enabled applications.

Trust

OPENi aims to inspire innovation in the European mobile applications industry, by radically improving the trust in personal cloud storage through the development of a consumer-centric, open source mobile cloud applications platform.

Mobile Client Libraries

In line with OPENi's ethos of openness and collabortation we have included a number of client libraries which greatly reduce the development overhead of OPENi enabled applications.

Getting started: Building an OPENi enabled app

To start building OPENi enabled apps follow the tutorials below. First, however, we recommend you read the Dashboard and Client Libraries subsections to familiarise yourself with the Core Platforms user and developer facing GUIs. If you just can't wait to get started you can download the sample app provided and start tinkering with the code.

Dashboards

The OPENi core platform comes with a number of dashboards, both for the end user and the developer, which simplify their interaction with the core platform.

  • User Dashboard: The User Dashboard allows the end user to view the data on their Cloudlets, adjust their permissions settings, and view analytics on how often their data was accessed.
  • Admin Dashboard: The Admin Dashboard allows developers to create App and Service Enabler Clients on the platform; it also allows them to associate permissions with their clients (permissions which are then presented to users when requesting access to their data/Cloudlet). The Admin dashboard also allows the app developer to create custom types, setup subscriptions, view data, and run aggregator queries.
  • API Builder Dashboard: The API builder enables the developer community to enhance and expand the Graph API.
